# may-the-force-be-with-you
Install this package, and the Force will be with you. Always.

## Installation

``` bash
$ npm install may-the-force-be-with-you
```

## Usage

``` js
var may = require('may-the-force-be-with-you');

may("theForce").beWith("you");
```

## Test

npm test
